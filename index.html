Memory in Motion

Memory in Motion is a take on the classic memory game built in jQuery. The player wins by matching all like user pairs before they run out of turns. This project strengthened my ability to manipulate the DOM and integrating third party JavaScript libraries like HandlebarsJS and MomentJS. Styles and animations were done with SCSS. The pixel art gifs used for the cards are by the exceedingly talented Glauber Kotaki at Counterplay Games.
